Atlas is now available as a public beta on iOS via TestFlight.

This initial release includes:

- **Private browsing** - No tracking or data collection by default
- **Built-in file storage** - Save and organize files directly in the browser
- **Secure browsing** - Protection against malicious sites and trackers

We're actively developing Atlas and would love your feedback. More updates coming soon.
